# MemeBrain :smirk_cat:
A local-first media browser with AI-powered search and organisation.

Browse, search and organise your local image collection without uploading your files to the cloud.

Designed for people whose folders look like this:

Philosophy
Local first
Offline by default
You own your data
Bring your own AI model
No telemetry
No hidden uploads
Everything should be inspectable
Content agnostic (yes, *those* folders work too...:expressionless:)
