# MemeBrain 😼

A local-first media browser with AI-assisted search and organisation.

Browse, search and organise your local image collection without uploading your files to the cloud.

**Search by filenames, tags, or what's actually in your images using local AI models.**

**Generate filenames based on what's in your images.**

Designed for people whose folders look like this:

```text
Memes/
├── Reaction Images/
├── Downloads/
├── New Folder (7)/
├── Untitled.png
├── Untitled (2).png
└── IMG_0835.PNG
```

## Philosophy

- 🖥️ Local first
- 🔒 Offline by default
- 👤 You own your data
- 🤖 Bring your own AI model
- 🚫 No telemetry
- ☁️ No hidden uploads
- 🔍 Everything should be inspectable
- 😑 Content agnostic *(yes, **those** folders work too...)*
